### 당일 공부한 것을 키워드 위주로 정리해서 올리는 Repo입니다.

- 길지 않게 키워드 위주로 올리기
- 설명이 필요하다면 길지않은 문장으로 간결하게 설명하기
- 말보다는 코드로 설명하기
- commit convention을 유지하기


---

### commit convention

- feat: 새로운 기능 추가
- fix: 버그 수정 및 각종 코드 및 파일 수정ㅎ
- docs: 문서 추가 및 수정
- style: 코드 포맷팅 및 코드의 내용 변경이 없는경우
- refactor: 코드 리펙토링
- test: 테스트 코드
